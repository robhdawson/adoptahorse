adopt a horse

A little API that will give you some horses.

This request:

GET https://adoptahorse.herokuapp.com/horse

Gives you something like:

{ "name": "Gorg", "mood": "cool", "talents": [ "contact juggling", "penmanship", "always gets the coke/pepsi taste test wrong" ], "id": "49740303261205554" }

You can also request multiple horses, like this:

GET https://adoptahorse.herokuapp.com/horses?n=2

They come back in an array like this:

[ { "name": "Endless Screaming", "mood": "real sad", "talents": [ "mapmakery" ], "id": "17178360698744655" }, { "name": "Big Boy", "mood": "sedated", "talents": [ "mapmakery", "knows the whole alphabet" ], "id": "015191152226179838" }, { "name": "Grunk", "mood": "revved up", "talents": [ "speechifying", "karate" ], "id": "4518112021032721" } ]

CORS is enabled, so fetch these beauties from anywhere you want.
